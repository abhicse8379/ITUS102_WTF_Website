Detailed Syllabus

30 hours, six units

Percentages indicate the approximate weight of each unit, used for balancing internal exam sets across units.

↓ Download syllabus PDF
01

Web Architecture Fundamentals

4 Hrs · 12%

Web & internet basics, client–server architecture, web development life cycle, web services & REST APIs, HTTP requests/responses/methods/status codes, monolithic vs microservice architecture, session management, stateful vs stateless apps, client vs server data storage, responsive design principles, CSS cascading & inheritance, Git & GitHub.

02

Client-Side Scripting Fundamentals

8 Hrs · 28%

JavaScript & the browser execution environment, rendering process, variables/data types/operators, conditionals, loops, functions, event handling, single-threaded execution, event queue & event loop, DOM introduction & manipulation, form validation, client-side data handling, async programming, timers & callbacks, dynamic page updates, browser storage, resource loading & performance, secure input handling.

03

Server-Side Processing Fundamentals

6 Hrs · 20%

Server-side processing intro, request–response lifecycle, dynamic content generation, execution models, data access layers, database connectivity & connection management, input validation & sanitization, authentication & authorization, session/token-based auth, role-based access control, concurrent requests, resource management, external service integration, sync vs async communication.

04

Web–Data Interaction Fundamentals

4 Hrs · 13%

Databases in web apps, data access interfaces, request/response flow with databases, basic query processing, CRUD operations, query optimization awareness, caching fundamentals, data consistency, handling connection failures, concurrent data access, connection-oriented vs connection-less communication.

05

Web Security Threats & Defense Awareness

4 Hrs · 13%

Web security introduction, common vulnerabilities, secure password practices, session security, timeouts & retries, input validation, secure coding awareness, defensive design, web hacking techniques & ethics, safe browsing, security best practices.

06

Modern Web Ecosystem

4 Hrs · 14%

Cross-browser compatibility, browser-specific issues, DevTools for debugging & inspection, mobile-friendly design, responsive testing, web scraping concepts & ethics, modern framework overview & selection criteria, scalability & performance awareness, industry trends.
